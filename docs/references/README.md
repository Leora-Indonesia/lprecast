# References Index

Folder ini berisi hasil olahan dari source material mentah yang disimpan di `docs/reference_file/`.

## Purpose

- menyediakan ringkasan cepat yang searchable
- memudahkan ekstraksi rule, KPI, dan dampak produk dari PDF mentah
- menjadi jembatan sebelum insight diserap ke dokumen aktif

## Relationship with `reference_file/`

- `docs/reference_file/` = raw upload only
- `docs/references/` = markdown summary hasil analisa raw upload

## Files

- `vendor-approval-checklist.md` - checklist approval vendor hasil normalisasi dari source kerja internal
- `cerita-nyata-spv-summary.md` - narrative journey SPV
- `end-to-end-scenario-spv-summary.md` - requirement detail flow SPV
- `simulasi-dana-summary.md` - skema deposit dan termin pembayaran
- `sop-klien-summary.md` - SOP formal client journey
- `sop-vendor-summary.md` - SOP formal vendor lifecycle

## Usage

1. baca summary di folder ini dulu
2. buka file mentah di `docs/reference_file/` jika perlu detail penuh
3. serap rule yang sudah disepakati ke `CONTEXT.md`, `modules/*.md`, atau doc flow terkait
