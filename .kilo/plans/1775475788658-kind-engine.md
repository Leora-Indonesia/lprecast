# Set Default Theme to Light

## Current State

The default theme is currently set to `"system"` in `components/theme-provider.tsx:13`:

```tsx
defaultTheme = "system"
```

## Change Required

Change `defaultTheme` from `"system"` to `"light"` in `components/theme-provider.tsx`:

```tsx
defaultTheme = "light"
```

## File to Edit

- `components/theme-provider.tsx` (line 13)

## Impact

- Users will see the light theme by default on first visit
- The theme can still be toggled via the `d` keyboard hotkey
- The theme preference is stored in localStorage by `next-themes`, so returning users will see their previous choice
