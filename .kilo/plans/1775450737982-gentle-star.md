### 3. Next.js Config Update

**File: `next.config.ts`** (atau `next.config.js`)

Tambahkan domain Unsplash ke allowed image domains:

```typescript
const config = {
  images: {
    remotePatterns: [
      {
        protocol: "https",
        hostname: "images.unsplash.com",
      },
    ],
  },
}
```
