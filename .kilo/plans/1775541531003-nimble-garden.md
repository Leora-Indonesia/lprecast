# Plan: Create lib/types/vendor.ts - TypeScript Types for Vendor Entities

## Overview

Derive TypeScript types for vendor-related entities from the existing `types/database.types.ts` using the `Database['public']['Tables']['table_name']['Row']` pattern.

The following entities will be derived from these database tables:

| Entity               | Database Table             |
| -------------------- | -------------------------- |
| VendorRegistration   | `vendor_registrations`     |
| VendorCompanyInfo    | `vendor_company_info`      |
| VendorContacts       | `vendor_contacts`          |

## Implementation

Create file `lib/types/vendor.ts` with:

1. Import Database type from `types/database.types.ts`
2. Export type aliases using pattern:
   ```typescript
   export type VendorRegistration =
     Database["public"]["Tables"]["vendor_registrations"]["Row"]
   ```
3. Include Insert and Update variants where applicable:
   ```typescript
   export type VendorRegistrationInsert =
     Database["public"]["Tables"]["vendor_registrations"]["Insert"]
   export type VendorRegistrationUpdate =
     Database["public"]["Tables"]["vendor_registrations"]["Update"]
   ```

## Tasks

1. Create directory `lib/types/` if not exists
2. Create file `lib/types/vendor.ts` with all type definitions
3. Run `pnpm typecheck` to verify type correctness
