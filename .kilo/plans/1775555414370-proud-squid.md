# Perbaikan Styling/Component pada Vendor Register Page

## Masalah yang Ditemukan

### 1. Font Awesome Icons Tidak Di-import (Critical)

**Lokasi:**

- `app/(vendor)/vendor/register/page.tsx` (line 38-43)
- `components/vendor/register/operational-form.tsx` (line 204, 247, 266, 297)

**Deskripsi:**
Menggunakan Font Awesome icons dengan class string (`fa-solid fa-*`) tanpa import Font Awesome library.

**File Terkait:**

- `page.tsx`: STEPS array mendefinisikan icon strings yang tidak digunakan
- `operational-form.tsx`: Inline `<i>` elements dengan Font Awesome classes

### 2. Inkonsistensi Icon Library

**Deskripsi:**

- File utama menggunakan Lucide React icons (`<Check>` dari lucide-react)
- operational-form.tsx menggunakan Font Awesome inline classes
- Harusnya konsisten menggunakan satu library (Lucide React)

### 3. CSS Class `required-star` Tidak Terdefinisi

**Lokasi:** Semua form components (company-info-form, legal-documents-form)
**Deskripsi:** Class `required-star` digunakan untuk asterisk wajib tapi perlu diperiksa keberadaannya

## Rencana Perbaikan

### Step 1: Perbaiki Icons di `operational-form.tsx`

Ganti Font Awesome icons dengan Lucide React icons:

**Line 204 (empty state icon):**

```tsx
// Sebelum:
;<i className="fa-solid fa-box-open mb-2 block text-4xl" />

// Sesudah:
import { Package } from "lucide-react"
;<Package className="mb-2 h-10 w-10 text-gray-300" />
```

**Line 247 (edit button):**

```tsx
// Sebelum:
;<i className="fa-solid fa-pen" />

// Sesudah:
import { Pencil } from "lucide-react"
;<Pencil className="h-4 w-4" />
```

**Line 266 (info icon):**

```tsx
// Sebelum:
;<i className="fa-solid fa-circle-info mr-1" />

// Sesudah:
import { Info } from "lucide-react"
;<Info className="mr-1 h-3 w-3" />
```

**Line 297 (map marker):**

```tsx
// Sebelum:
;<i className="fa-solid fa-map-marker-alt text-xs" />

// Sesudah:
import { MapPin } from "lucide-react"
;<MapPin className="h-3 w-3" />
```

### Step 2: Perbaiki STEPS Array di `page.tsx`

Hapus atau update property `icon` di STEPS array karena tidak digunakan:

**Opsi A:** Hapus property icon (recommended karena tidak digunakan)

```tsx
const STEPS = [
  { label: "Informasi Perusahaan" },
  { label: "Dokumen Legal" },
  { label: "Operasional" },
  { label: "Review & Submit" },
]
```

### Step 3: Verifikasi CSS Class `required-star`

Periksa apakah class ini ada di global styles. Jika tidak, tambahkan:

```css
.required-star {
  color: #ef4444;
  margin-left: 2px;
}
.required-star::before {
  content: "*";
}
```

Atau gunakan inline styling/Tailwind:

```tsx
<span className="ml-0.5 text-red-500">*</span>
```
