# Plan: Install Context7 MCP di Kilo

## Goal

Pasang Context7 MCP server di Kilo dengan menyalin konfigurasi dari Opencode.

## Sumber Setup (Opencode)

File: `/home/alv-linux/.config/opencode/opencode.json`

```json
{
  "mcp": {
    "Context7": {
      "type": "remote",
      "url": "https://mcp.context7.com/mcp",
      "headers": {
        "CONTEXT7_API_KEY": "ctx7sk-d0009fd4-9559-46bc-bc39-3ef7c98ff7f7"
      }
    }
  }
}
```

## Langkah

1. **Tambah konfigurasi MCP ke kilo.jsonc**

   Edit file `/home/alv-linux/.config/kilo/kilo.jsonc` — tambahkan field `mcp` dengan konfigurasi Context7:

   ```jsonc
   {
     "$schema": "https://app.kilo.ai/config.json",
     "permission": { ... },
     "mcp": {
       "context7": {
         "type": "remote",
         "url": "https://mcp.context7.com/mcp",
         "headers": {
           "CONTEXT7_API_KEY": "ctx7sk-d0009fd4-9559-46bc-bc39-3ef7c98ff7f7"
         }
       }
     }
   }
   ```

2. **Verifikasi** — Restart Kilo dan cek apakah MCP terhubung.

## Catatan

- Sama seperti Opencode, menggunakan remote MCP di `https://mcp.context7.com/mcp`
- API key yang digunakan sama: `ctx7sk-d0009fd4-9559-46bc-bc39-3ef7c98ff7f7`
