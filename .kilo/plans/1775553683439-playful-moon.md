# Plan: Fix Checkmark Icon Not Showing / Not Filled

## File: `/Users/alvinstuff/lprecast/components/vendor/register/stepper.tsx`

### Issue

Line 50: `<Check className="h-5 w-5" />` - Check icon not visible or not filled

### Root Cause

The `Check` icon from lucide-react is outline by default. Need to add `fill="currentColor"` to make it solid/filled.

### Fix

```typescript
// Before (line 50)
<Check className="h-5 w-5" />

// After
<Check className="h-5 w-5" fill="currentColor" />
```

The `fill="currentColor"` will use the `text-white` color from the parent div (bg-green-500 text-white), making the checkmark appear as a solid green checkmark.
