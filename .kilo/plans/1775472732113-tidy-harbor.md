# Plan: Tambahkan Shortcut `pnpm db:pull` dan `pnpm db:push`

## Tujuan

Membuat shortcut npm scripts untuk generate TypeScript types dari Supabase database.

## Kesimpulan dari Context7

### Perintah yang Digunakan

**`supabase gen types typescript`** - Generate TypeScript types dari database schema

Contoh penggunaan:

```bash
npx supabase gen types typescript --project-id "$PROJECT_REF" --schema public > types/database.types.ts
```

Flags yang relevan:

- `--linked` - Generate types dari linked project (butuh Docker)
- `--project-id` - Generate types dari project ID langsung
- `--db-url` - Generate types dari connection string langsung
- `--schema` - Specify schema (default: public)

## Implementasi

### 1. Update `package.json`

Tambahkan scripts:

```json
{
  "scripts": {
    "db:pull": "npx supabase gen types typescript --project-id phfuwunwgzkfzettekkh --schema public > types/database.types.ts",
    "db:push": "npx supabase db push"
  }
}
```

### 2. Buat folder `types/`

```bash
mkdir -p types
```

## Pertanyaan untuk User

Berdasarkan request "untuk ambil db types dari supabase", ada ambiguitas:

1. Apakah maksudnya **generate TypeScript types** dari database schema? Jika ya, perintah yang benar adalah `supabase gen types typescript`, bukan `db pull/push`.

2. Atau apakah memang ingin menggunakan **`db pull`** (untuk pull schema ke migration) dan **`db push`** (untuk push migration ke remote)?

## Rekomendasi Implementasi

Jika tujuannya **menghasilkan TypeScript types**, saya sarankan:

```json
{
  "db:pull": "supabase gen types typescript --linked --schema public > types/database.types.ts",
  "db:push": "supabase db push"
}
```

Dimana:

- `db:pull` â†’ Generate TypeScript types dari remote database (melalui project yang sudah di-link)

## Langkah Implementasi

1. **Tambahkan scripts ke `package.json`:**
   - `db:pull` - Generate types dari linked project
   - `db:push` - Push migrations ke remote

2. **Buat folder `types/`** jika belum ada untuk menyimpan generated types

3. **Pastikan Supabase CLI terinstall** (sudah ada di `supabase/.temp/cli-latest`)

## Catatan

- Project perlu di-link dulu dengan `npx supabase link` agar `db:push` dan `db:pull` dengan `--linked` bekerja
- Untuk `--linked` bekerja, perlu Docker running
- Alternatively, bisa pakai `--project-id` langsung tanpa perlu link
