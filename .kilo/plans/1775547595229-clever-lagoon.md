# Plan: Tambahkan Sonner Toast Notifications

## Tujuan

Mengganti `alert()` dengan Sonner toast untuk notifikasi yang lebih baik di vendor registration form.

## Events yang Perlu Toast

### 1. Form Submission (page.tsx)

- **Success**: Pendaftaran berhasil → redirect ke success page
- **Error**: Gagal submit (server error / network error)

### 2. Draft Persistence (page.tsx)

- **Load draft**: Draft berhasil dimuat ke form
- **Clear draft**: Draft dihapus (user pilih "Mulai Baru")

## Implementation Steps

### Step 1: Install Sonner

```bash
pnpm add sonner
```

### Step 2: Create Toaster Component (if needed)

File: `components/ui/toaster.tsx` (optional, can also add directly to layout)

```tsx
"use client"

import { Toaster } from "sonner"

export function ToastProvider() {
  return <Toaster position="top-center" richColors />
}
```

### Step 3: Add Toaster to Root Layout

File: `app/layout.tsx`

```tsx
import { Toaster } from "sonner"

// Add inside <body>
;<Toaster position="top-center" richColors />
```

### Step 4: Update page.tsx

#### Import toast

```tsx
import { toast } from "sonner"
```

#### Replace alerts in handleSubmit

```tsx
// Success - sudah redirect ke success page, mungkin tidak perlu toast
// Tapi bisa tambahkan toast sebelum redirect

// Error
toast.error("Gagal mengirim data", {
  description: result.error || "Silakan coba lagi",
})
```

#### Add toast for draft operations

```tsx
// Dialog "Lanjutkan" button
onClick={() => {
  loadSavedData()
  toast.success("Draft berhasil dimuat")
  setShowLoadDialog(false)
}}

// Dialog "Mulai Baru" button
onClick={() => {
  clearSavedData()
  toast.info("Draft dihapus")
  setShowLoadDialog(false)
}}
```

## Toast Types to Use

| Event          | Type                         | Message                             |
| -------------- | ---------------------------- | ----------------------------------- |
| Submit success | `toast.success()`            | "Pendaftaran berhasil dikirim"      |
| Submit error   | `toast.error()`              | "Gagal mengirim data" + description |
| Load draft     | `toast.success()`            | "Draft berhasil dimuat"             |
| Clear draft    | `toast.info()`               | "Draft dihapus"                     |
| Auto-save      | `toast.loading()` (optional) | "Menyimpan..."                      |
