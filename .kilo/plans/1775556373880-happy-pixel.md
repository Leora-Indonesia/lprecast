# Plan: Replace Header in Vendor Register Page

## Goal

Replace the header in `/app/(vendor)/vendor/register/page.tsx` with a header matching the homepage style, but without any menu.

## Current State

- Register page has a custom header (lines 264-286) with "LPrecast Vendor Portal" branding and a Login button
- Homepage header is embedded inline (app/page.tsx lines 19-68) with logo, navigation menu, and login button

## Proposed Solution

Create a new header component that matches the homepage header styling (logo, sticky, backdrop-blur) but without the navigation menu.

## Steps

1. Create a new `VendorHeader` component at `/components/vendor/vendor-header.tsx`
   - Use same styling as homepage header (sticky, backdrop-blur, border-b)
   - Display only the logo/brand (no navigation menu)
   - Include Login button as in current register header

2. Update `/app/(vendor)/vendor/register/page.tsx`
   - Import `VendorHeader` component
   - Replace the inline `<nav>` block (lines 264-286) with `<VendorHeader />`

## Implementation

```tsx
            <span className="text-sm text-muted-foreground">
              Sudah punya akun?
            </span>
            <Button onClick={() => router.push("/login")}>Login</Button>
          </div>
        </div>
      </div>
    </header>
  )
}
```

### Update: `/app/(vendor)/vendor/register/page.tsx`
