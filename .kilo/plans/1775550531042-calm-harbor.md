# Plan: Redesign Vendor Registration Page

## Overview

Mengubah design halaman vendor registration dari step indicator vertikal menjadi horizontal tabs, dengan mempertahankan semua field yang ada.

## Constraints

- Gunakan warna theme yang sudah ada (primary, destructive, dll.)
- Tetap gunakan shadcn UI dan Tailwind CSS
- Label ditulis dengan kapital (Title Case), bukan UPPERCASE
- Tidak mengurangi jumlah form fields

## Tasks

### 1. Update Main Page Component (`app/(vendor)/vendor/register/page.tsx`)

**Step Indicator Changes:**

- Ubah step indicator dari circles dengan connecting line menjadi horizontal tabs
- Setiap tab memiliki icon dan label
- Tab yang sedang aktif: border-bottom green-500, background white
- Tab yang sudah complete: tampilkan checkmark icon
- Tab yang belum aktif: text gray-500, border transparent

**Navigation Buttons:**

- Ubah layout button navigation
- Button "CANCEL" hanya muncul di step 1
- Button "PREVIOUS" muncul di step 2-4
- Button "CONTINUE" berubah menjadi "SUBMIT" di step terakhir
- Hitung ulang step karena HTML referensi punya 4 step (sama dengan current)

### 2. Update Company Info Form (`components/vendor/register/company-info-form.tsx`)

**Layout Changes:**

- Section header style: text-lg font-semibold, border-bottom padding
- Grid layout: 2 kolom untuk input fields
- Label style: text-sm font-medium text-gray-800 (Title Case, bukan uppercase)
- Input styling: border rounded, focus state dengan green border
- Kontak Alternatif: tetap gunakan table layout dengan 3 rows (sesuai data yang ada)

### 3. Update Legal Documents Form (`components/vendor/register/legal-documents-form.tsx`)

**Layout Changes:**

- Setiap dokumen dalam card dengan border
- Upload button styling: green accent colors
- File input: custom styling dengan label dan input file

### 4. Update Operational Form (`components/vendor/register/operational-form.tsx`)

**Layout Changes:**

- Section dengan border-top separator
- Grid layout untuk field rekening
- Pertahankan semua sections: Bank info, Alamat Pabrik, Daftar Harga Produk, Area Pengiriman, Biaya & Inklusi

### 5. Update Review Submit (`components/vendor/register/review-submit.tsx`)

**Layout Changes:**

- Review sections dalam bordered boxes
- Checkbox dengan label yang lebih prominent
