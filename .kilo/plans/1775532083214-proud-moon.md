# Plan: Multi-Select Area Pengiriman Popup

## Goal

Tambahkan tombol "Tambah Area" pada section Area Pengiriman yang membuka popup/modal untuk memilih multiple wilayah (provinsi dan kabupaten/kota).

## Changes Required

### 1. Update Area Pengiriman Section

Replace existing province/kabupaten dropdowns with:

- Label "Area Pengiriman"
- Helper text: "Klik 'Tambah Area' untuk memilih wilayah pengiriman"
- A "Tambah Area" button that opens a popup modal

### 2. Create Area Selection Popup Modal

Create a new modal (`areaModal`) with:

- Header: "Pilih Area Pengiriman" with close button
- Search input for filtering
- Hierarchical display: Province → Kabupaten/Kota checkboxes
- "Pilih Semua" checkbox per province
- Selected areas displayed as removable tags
- Footer buttons: "Batal" and "Terapkan"

### 4. Add JavaScript for Area Modal

**`openAreaModal()`:**

- Show modal with `display: flex`
- Populate area tree from `provincesData`
- Show currently selected areas as checked

**`closeAreaModal()`:**

- Hide modal

**`applyAreaSelection()`:**

- Collect selected province/kabupaten IDs
- Update `selectedAreas` array
- Render selected areas as tags below button
- Close modal

**`renderSelectedAreas()`:**

- Update to display areas as removable tags (already exists, may need update for popup approach)

### 5. Data Structure

Selected areas stored as array of `{type: 'provinsi'|'kabupaten', id: number, name: string, province: string}`.

## UI Flow

1. User clicks "Tambah Area" button in Area Pengiriman section
2. Modal opens with searchable province → kabupaten tree
3. User selects provinces/kabupatens (checkboxes)
4. Click "Terapkan" → modal closes, selected areas shown as tags
5. Tags can be removed individually by clicking ×
